```mermaid
sequenceDiagram
    actor Creador
    actor Espectador
    participant API as MonetizacionEcosistemaCreador (5005)
    participant Aud as Audiencia (5004)
    participant DB as MySQL youtube_monetizacion

    Note over Creador,DB: 1) Evaluar elegibilidad (RF-M1)
    Creador->>API: POST /api/eligibility/evaluate {ChannelId, SubscriberCount, WatchHours}
    API->>Aud: GET /api/subscriptions/count?channelId= (suscriptores y horas)
    Aud-->>API: count / watchHours
    API->>API: IsEligible = Subs>=1000 && Hours>=4000
    API->>DB: upsert MonetizationEligibility (Status=Eligible/Rejected, EvaluatedAt)
    API-->>Creador: 200 EligibilityResponse

    Note over Creador,DB: 1b) Solicitar monetización del canal (RF-M1)
    Creador->>API: POST /api/monetization/applications {ChannelId, CreatorUserId, Notes?}
    API->>DB: insert MonetizationApplication (Status=UnderReview, RequestedAt)
    API-->>Creador: 201 MonetizationApplicationResponse

    Note over Creador,DB: 1c) Activar/desactivar monetización (canal o contenido) (RF-M1)
    Creador->>API: PUT /api/monetization/channels/{channelId}/status {Enabled, Reason?}
    API->>DB: update estado de monetización del canal
    API-->>Creador: 200 MonetizationStatusResponse
    Creador->>API: PUT /api/monetization/content/{catalogItemId}/status {Enabled, Reason?}
    API->>DB: update estado de monetización del contenido
    API-->>Creador: 200 MonetizationStatusResponse

    Note over Creador,DB: 2) Crear producto y registrar compra (RF-M2)
    Creador->>API: POST /api/products {ChannelId, Type, Name, Price}
    API->>DB: insert MonetizationProduct
    API-->>Creador: 201 ProductResponse
    Espectador->>API: POST /api/products/purchases {ProductId, ViewerUserId, Amount}
    API->>DB: insert ProductPurchase
    API-->>Espectador: 201 ProductPurchaseResponse

    Note over Creador,DB: 3) Registrar ingreso (RF-M3)
    Creador->>API: POST /api/revenue {ChannelId, VideoId?, Source, Amount, PlatformSharePercent}
    API->>API: calcular PlatformShare / CreatorShare
    API->>DB: insert RevenueEntry (Status=Pending/Confirmed/Paid)
    API->>DB: upsert CreatorEarnings (Balance += CreatorShare, UpdatedAt)
    API-->>Creador: 201 RevenueResponse

    Note over Creador,DB: 4) Consultar ganancias (RF-M4)
    Creador->>API: GET /api/earnings/{channelId}?from=&to=&groupBy=&format=
    API->>DB: leer CreatorEarnings y agregar RevenueEntry por source/video/periodo
    API-->>Creador: 200 EarningsResponse (con breakdown y rpm)

    Note over Creador,DB: 5) Documentación fiscal y payout (RF-M5)
    Creador->>API: POST /api/fiscal-documents {ChannelId, DocumentType, DocumentNumber, Country}
    API->>DB: insert FiscalDocument (Status=PendingReview)
    API-->>Creador: 201 FiscalDocumentResponse

    Creador->>API: POST /api/payouts {ChannelId, Amount, MinimumPayoutAmount, TaxDocumentId?}
    API->>DB: leer CreatorEarnings.Balance
    alt Amount > Balance OR Amount < MinimumPayoutAmount
        API-->>Creador: 409 CONFLICT
    else Amount <= Balance AND Amount >= MinimumPayoutAmount
        API->>DB: insert Payout (Status=Pending, RequestedAt)
        API->>DB: CreatorEarnings.Balance -= Amount
        API-->>Creador: 201 PayoutResponse (Pending)
    end

    Note over Creador,DB: 5b) Resolver payout
    Creador->>API: POST /api/payouts/{id}/complete
    API->>DB: update Payout (Status=Paid, PaidAt)
    API-->>Creador: 200 PayoutResponse (Paid)
    Creador->>API: POST /api/payouts/{id}/fail
    API->>DB: update Payout (Status=Failed)
    API->>DB: CreatorEarnings.Balance += Payout.Amount (reintegro)
    API-->>Creador: 200 PayoutResponse (Failed)
```
