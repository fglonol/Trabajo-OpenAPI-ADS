```mermaid
sequenceDiagram
    actor Anunciante
    participant API as Publicidad API
    participant Player as Reproductor (cliente)
    participant Mon as Monetizacion API (5005)

    Note over Anunciante,API: Onboarding de anunciante (RF-F1)
    Anunciante->>API: POST /api/advertisers (Name, Email, BillingName, BillingAddress, TaxId, PaymentMethodToken)
    API-->>Anunciante: 201 Advertiser

    Note over Anunciante,API: Estimar inventario antes de presupuestar (RF-F4/RF-F5)
    Anunciante->>API: POST /api/inventory/estimate (placement, targeting, brandSafetyLevel)
    API-->>Anunciante: 200 { estimatedImpressions, eligibleCatalogItems, estimatedCpm }

    Note over Anunciante,API: Crear y activar campaña
    Anunciante->>API: POST /api/campaigns (Budget, fechas)
    API-->>Anunciante: 201 Campaña (Draft)
    Anunciante->>API: POST /api/campaigns/{id}/creatives (incluye policyReviewStatus)
    Anunciante->>API: PUT /api/campaigns/{id}/targeting (incluye contentCategories, excludedCatalogItemIds)
    Anunciante->>API: POST /api/campaigns/{id}/activate
    API-->>Anunciante: 200 Campaña (Active)

    Note over Anunciante,API: Pausar y reanudar (RF-F2)
    Anunciante->>API: POST /api/campaigns/{id}/pause
    API-->>Anunciante: 200 Campaña (Paused)
    Anunciante->>API: POST /api/campaigns/{id}/resume
    API->>API: Verifica Status=Paused y Budget>Spend
    API-->>Anunciante: 200 Campaña (Active) (o 409 CONFLICT)

    Note over Player,API: Entrega de anuncio (RF-F6)
    Player->>API: GET /api/ads/serve?placement&country&interest
    API->>API: Selecciona campaña Active con Budget>Spend, targeting compatible y creativo Approved
    API-->>Player: 200 { campaignId, creative, brandSafetyDecision } (o 404 NO_FILL)

    Note over Player,API: Medición (RF-F7)
    Player->>API: POST /api/measurement/events (Impression, catalogItemId)
    API->>API: Spend += CostPerEvent; si Spend>=Budget => Completed
    Player->>API: POST /api/measurement/events (Click)
    Player->>API: POST /api/measurement/events (Skip)

    Note over API,Mon: Liquidación de ingresos al creador
    API-)Mon: POST http://localhost:5005/api/revenue (Source=Ads, Amount)
    Mon-->>API: 201 (ingreso registrado)

    Note over Anunciante,API: Reporting y facturación (RF-F8)
    Anunciante->>API: GET /api/reporting/campaigns/{id}
    API-->>Anunciante: 200 { impressions, clicks, skips, ctr, cpm, spend, remainingBudget, invoiceId }
    Anunciante->>API: GET /api/reporting/invoices/{advertiserId}?from&to
    API-->>Anunciante: 200 PagedResult~InvoiceResponse~
```
